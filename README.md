### `About`

국립국어원의 한국어 맞춤법/문법 검사기의 lite 버전을 만들어보려고 한다.\
기존 서비스가 정말 좋지만 사용하는 데 아쉬운 부분 두 가지가 있는데,\
1 일분일초가 귀한 사람에게능 너무너무 느리다.\
2 맞춤법 검사를 하는 도중에 오류가 나거나 검사 가능한 글자 수를 넘어도 오류가 난다.\
